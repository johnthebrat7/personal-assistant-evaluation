# Qwen2.5 Personal Assistant

Open-source personal assistant (Qwen2.5-0.5B-Instruct) running on free CPU.
Features: multi-turn memory, rule-based safety guardrails, calculator/date tools,
and per-session observability (turns, blocks, tool uses, average latency).

Part of an OSS-vs-frontier evaluation project. Self-hosted on Hugging Face Spaces.
